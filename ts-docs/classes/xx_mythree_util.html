Static makeExtrudeGeometry

  • makeExtrudeGeometry(shape: THREE.Shape, amount: number): THREE.ExtrudeBufferGeometry
    • Defined in sources/media-webgl/three-js/demos/demo2_mini_city/js/utils.ts:56

    模子 + 数量 ————> 拉伸几何形 TIP 网上资料不是很多：大致上，是将一个二维图形拉伸成三维图形。

    参考资料： three.js中几何对象 - CircleGeometry 、 CubeGeometry、CylinderGeometry 和 ExtrudeGeometry- 【SegmentFault 思否】 - https://segmentfault.com/a/1190000012682069 在线做实验 - https://threejs.org/docs/#api/en/geometries/ExtrudeBufferGeometry

    经过多次实验，得出以下理解： 1.以一个树木的2D轮廓为基础。 2.反复调试【ExtrudeBufferGeometry】的各种参数：参数的调节，就像增加缓坡和缓冲。让物体变厚，让物体边缘变的自然而柔和。 3.直到，勉强像一颗少儿动画里的树一样。（就是个调参的过程。）

    Parameters

    • shape: THREE.Shape
    • amount: number

    Returns THREE.ExtrudeBufferGeometry
